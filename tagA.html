TAG "A"

  • Tag "A" também conhecida como âncora é responsavel por vincular os demais links internos ou externos ao HTML.

  • Outro atributo importante da tag "A" é o href (hypertext refence), é o local onde se coloca a sua URL.

  • O target que também é um atributo da tag "A", ele é usado para decidir se um URL quando aberto, irá direcionar para uma nova aba ou abri-rá no mesmo site.


    ° Para abrir na mesma aba use o comando ' target ="blank" ' após inserir o link;

    ° Para abrir em outra aba use o comando ' target ="blank" ' após inserir o link
